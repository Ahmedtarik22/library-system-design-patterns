# Library Management System

Simple Java Library Management System using basic Design Patterns:
- **Singleton** (LibraryService)
- **Factory** (BookFactory)
- **Decorator** (PremiumBookDecorator)
- **Chain of Responsibility** (Librarian/Manager Handlers)
- **Facade** (LibraryFacade)
- **Adapter** (BookAdapter)
- **Observer** (User Notifications)

## Project Structure
The project is organized into clear packages for each pattern:
- `model`: Core book and user classes.
- `service`: Central management (Singleton).
- `factory`: Object creation logic.
- `decorator`: Dynamic feature additions.
- `chain`: Approval process.
- `facade`: Simplified system interface.
- `adapter`: Legacy data support.
- `observer`: Notification system.

## How to Run
Compile and run the `Main.java` file in the `src` directory.
